# Deploying Changes to gibraltarbat.com

After making any changes to the site:

1. Open Terminal in this folder (right-click the `gibraltar-bat` folder in Finder → "New Terminal at Folder")

2. Run these three commands:

```
git add .
git commit -m "describe what changed"
git push
```

3. Vercel auto-deploys within ~30 seconds. Done.
